# izzy-and-the-isolators

## Set up

- ✅ you must have NodeJS and Git installed
  - 👉 https://nodejs.org
  - 👉 https://git-scm.com
- 📝 clone repository with `git clone https://github.com/xav-ie/izzy-and-the-isolators.git`
- 📂 `cd izzy-and-the-isolators`
- 🦄 `npm install`
- ✨ `node app.js`
- 🌎 visit http://localhost:5555
